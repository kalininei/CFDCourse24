5 # Подготовка {#prep}
6 1. Для сборки проекта необходимо установить git и cmake>=3.0
7  - В Windows необходимо скачать и установить диструбутивы:
8  - https://github.com/git-for-windows/git/releases/download/v2.38.1.windows.1/Git-2.38.1-64-bit.exe
9  - https://github.com/Kitware/CMake/releases/download/v3.24.2/cmake-3.24.2-windows-x86_64.msi
10 
11  При установке cmake проследите, что бы путь к cmake.exe сохранился в системных путях. Msi установщик спросит об этом в диалоге.
12  - В линуксе используйте менеджеры пакетов, предоставляемые вашим дистрибутивом.
13  Также проследите чтобы были доступны компиллятор g++ и отладчик gdb.
14 
15 2. Создайте папку в системе для репозиториев. Например D:/git_repos/
16 
17 3. Возьмите необходимые заголовочные библиотеки boost из https://disk.yandex.ru/d/GwTZUvfAqPsZBQ
18  и распакуйте архив в папку для репозиториев (D:/git_repos/boost).
19  Проследите, чтобы внутри папки boost сразу шли папки с кодом (accumulators, algorithm, ...) и заголовочные файлы (align.hpp, aligned_storage.hpp, ...)
20  без дополнительных уровней вложения.
21 
22 4. Откройте терминал (git bash в Windows).
23 
24 5. С помощью команды cd в терминале перейдите в папку для репозиториев
25  > cd D:/git_repos
26 
27 6. Клонируйте репозиторий
28  > git clone https://github.com/kalininei/CFDCourse24
29  В директории (D:/git_repos в примере) появится папка CFDCourse24, которая является корневой папкой проекта
30

32 # VisualStudio {#vs}
33 
34 1. Cоздайте папку build в корне проекта СFDCourse24
35 
36 2. Скопируйте скрипт winbuild64.bat в папку build. Далее вносить изменения
37  только в скопированном файле.
38 
39 3. Скрипт написан для версии Visual Studio 2019. Если используется другая версия,
40  измените в скрипте значение переменной CMGenerator на соответствующие вашей версии.
41  Значения для разных версий Visual Studio написаны ниже
42 ```
43 SET CMGenerator="Visual Studio 17 2022"
44 SET CMGenerator="Visual Studio 16 2019"
45 SET CMGenerator="Visual Studio 15 2017"
46 SET CMGenerator="Visual Studio 14 2015"
47 ```
48 
49 4. Запустите скрипт winbuild64.bat из папки build. Нужен доступ к интернету.
50  В процессе будет скачано около 200Мб пакетов, поэтому первый запуск может занять время
51 
52 5. После сборки в папке build появится проект VisualStudio cfdcourse24.sln.
53  Его нужно открыть в VisualStudio.
54  Дерево решения должно иметь следующий вид.
55  ![](vs_solution_explorer.png)
56  Проекты:
57  - cfd24 - расчётная библиотека
58  - cfd24_test - модульные тесты для расчётных функций
59 
60 6. Проект cfd24_test необходимо назначить запускаемым проектом. Для этого нажать правой кнопкой мыши по проекту и в выпадающем меню
61  выбрать соответствующий пункт. После этого заголовок проекта должен стать жирным.
62 
63 7. Скомпиллировать решение. Несколько способов:
64  - `Ctrl+Shift+B`,
65  - `Build->Build Solution` в основном меню,
66  - `Build Solution` в меню решения в дереве решения,
67  - `Build` в меню проекта cfd24_test.
68 
69 8. Запустить тесты (проект cfd24_test) нажав F5 (или кнопку отладки в меню).
70  После отработки должно высветиться сообщение об успешном прохождении всех тестов.

72 # VSCode {#vscode}
73 
74 1. Открыть корневую папку проека через File->Open Folder
75 2. Установить предлагаемые расширения cmake, c++
76 3. Для настройки отладки создайте конфигурацию launch.json слудующего вида
77  ![](vscode_launch_json.png)
78  - Для этого перейдите в меню 'Run and Dubug' (Ctrl+Shift+B), нажмите
79  create launch.json, выберите пункт Node.js.
80  - После этого в корневой папке появится файл .vscode/launch.json.
81  - Откройте этот файл в vscode, нажмите Add configuration, Launch.
82  - Далее напишите имя программы как показано на картинке.
83  - Используйте поле args для установки аргументов запуска.
84  - Выберите созданную конфигурацию для запуска отладчика по F5
85  ![](vscode_launch.png)
